スマホ、PC対応1~4人用カードゲーム
5ステージくらいの敵を倒していくゲーム
キャラクターごとにデッキ固定

【カードのシステムについて】
カードのコスト＝AP消費量。
同カードは3つまで重複可。
毎ターン手札に3枚ランダムに補充。
味方への効果は自分へも発動可能。
使用したカードは手札から消える

【敵キャラクター案】
全ての敵はHPに人数×30加算(例:スライムは一人用の時HP100、二人用の時HP130)
特殊能力は常に敵の右上に表示される
スライム：HP 70　攻撃力5
特殊能力：なし
ゴブリン：HP 90　攻撃力7　
特殊能力：体力の少ないプレイヤーを優先的に攻撃する
オーク：HP 100　攻撃力9
特殊能力：攻撃力を2低下させる
フギン：HP60　攻撃力6
特殊能力：3ターンに一度味方(ムニン)のHPを10回復する、体力の少ないプレイヤーを優先的に攻撃する。
ムニン：HP60　攻撃力6
特殊能力：3ターンに一度味方(フギン)のHPを10回復する、体力の多いプレイヤーを優先的に攻撃する。
魔王：HP120　攻撃力10
特殊能力：3ターンに一度分身を二体発生させる。一度攻撃されると消える

【ステージ構成】
ステージが終わるごとにHPが10回復する

ステージ１：スライム
ステージ２：ゴブリン
ステージ３：オーク
ステージ４：フギン＆ムニン(二体)
ステージ５：魔王

【プレイアブルな方のキャラクター案】
全キャラクターのAP(アクションポイント)は7固定。毎ターンごとに補充される。カードのコスト＝AP消費量

剣士：アタッカー
HP 50

魔法使い：バッファー＋ちょいアタッカー
HP 40

僧侶：ヒーラー
HP 30
